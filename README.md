# 📱 صحتي — تطبيق Android

تطبيق Flutter لمتابعة السكر والضغط مع منبه حقيقي بصوت الهاتف.

## 🚀 كيفية البناء عبر GitHub

### الخطوة 1 — أنشئ Repository جديد
1. اذهب إلى github.com
2. اضغط **New repository**
3. الاسم: `sehati-app`
4. اختر **Public**
5. اضغط **Create repository**

### الخطوة 2 — ارفع هذه الملفات
1. اضغط **uploading an existing file**
2. ارفع **كل الملفات** من هذا المجلد

### الخطوة 3 — انتظر البناء
1. اذهب إلى تبويب **Actions**
2. انتظر الـ workflow ينتهي (10-15 دقيقة) ✅

### الخطوة 4 — حمّل APK
1. اضغط على الـ workflow المكتمل
2. اضغط **Artifacts**
3. حمّل **sehati-apk**
4. فك الضغط — احصل على `app-arm64-v8a-release.apk`

### الخطوة 5 — ثبّت على هاتفك
1. انقل APK لهاتفك
2. افتحه — اضغط **تثبيت**
3. إذا ظهر تحذير: **إعدادات ← أمان ← مصادر غير معروفة** ← فعّل

## ✅ مميزات التطبيق
- 🔔 **منبه حقيقي** بصوت الهاتف حتى الشاشة مغلقة
- 🩸 تسجيل السكر بوحدة g/L
- 💉 تسجيل الضغط بوحدة cmHg
- 📅 مواعيد قياس يومية قابلة للتعديل
- 💾 حفظ البيانات محلياً بدون إنترنت
- 👤 ملف المريض الشخصي

## 📋 متطلبات
- Android 6.0 (API 23) أو أحدث
- أي هاتف Android
